# New Project Prompt

Create or scaffold a Flutter project for a Studio Ghibli catalog app intended to be operated with Antigravity workflows.

## Stack
- Flutter
- Material 3
- Riverpod
- go_router
- Dio
- Easy Localization
- local persistence for cache, favorites, and recently viewed

## Required app scope
- film list
- film detail
- favorites
- recently viewed
- cache-first rendering

## Required repository support
- honor `AGENTS.md`
- honor shared rules under `agents/shared/`
- keep Antigravity workflows usable
- align implementation to `design/DESIGN.md`

## Architecture
Use feature-first structure with `presentation`, `domain`, and `data` layers.

## Output expectations
- app shell
- films feature baseline
- favorites baseline
- recently viewed baseline
- localization-ready strings
- verification results
