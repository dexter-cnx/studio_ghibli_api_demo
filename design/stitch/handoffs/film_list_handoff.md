# Film List Handoff

## Source
- Stitch screen: film list
- Design baseline: `design/DESIGN.md`

## Implement in Flutter with
- Material 3
- Riverpod
- go_router
- localized strings
- cache-first rendering

## UI requirements
- top search bar
- compact sort control
- vertically stacked film cards
- favorite toggle on each card
- loading, empty, error, and stale-cache states

## Notes
Use the design as a layout contract, not as a source of Flutter code structure.
