# Recently Viewed Handoff

## Purpose
Help users reopen films they recently viewed.

## Layout
1. app bar
2. compact recent film list or empty state

## Components
- app bar
- compact list items or cards
- empty state block

## Data Requirements
- title
- original title
- release year
- recency cue

## States
- success
- empty

## Implementation Notes
- local persistence only
- compact presentation preferred
