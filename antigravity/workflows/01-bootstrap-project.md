# Workflow: Bootstrap Project

## Goal
Create the initial Flutter app shell for the Studio Ghibli catalog project.

## Inputs
- existing repository state
- `design/DESIGN.md`
- `design/stitch/product_brief.md`
- `design/stitch/screen_specs.md`

## Steps
1. Inspect the current repo structure and toolchain.
2. Set up app shell: router, theme, localization bootstrapping, dependency wiring.
3. Establish feature-first structure for `films`, `favorites`, and `recently_viewed`.
4. Add API client and endpoint definitions.
5. Add local persistence scaffolding for cache and local user state.
6. Add placeholder routes/screens if full features are not yet implemented.
7. Run verification.
8. Produce implementation and verification artifacts.
