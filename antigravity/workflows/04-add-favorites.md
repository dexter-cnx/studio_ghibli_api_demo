# Workflow: Add Favorites

## Goal
Add local-only favorites that can be toggled from list and detail screens.

## Steps
1. Inspect local persistence conventions.
2. Create or extend local favorites data source.
3. Add repository contract and implementation if needed.
4. Add use cases or provider actions for read/toggle operations.
5. Integrate favorite state into list and detail UI.
6. Add favorites page and route if missing.
7. Add tests for toggle and persistence behavior.
8. Run verification and write artifacts.
