# Google Stitch Steps for This Project

## Suggested order
1. update `design/stitch/product_brief.md`
2. update `design/stitch/screen_specs.md`
3. use the prompts in `design/stitch/stitch_prompts.md`
4. refine film list and film detail first
5. ask Stitch to summarize each stable screen as a handoff draft
6. save the results under `design/stitch/handoffs/`
7. update `design/DESIGN.md` with stable design rules
8. begin implementation with repo prompts

## Good prompt pattern
- broad app concept first
- then one screen at a time
- then state patterns
- then handoff summaries

## Handoff reminder
A good handoff should include:
- purpose
- layout
- components
- data requirements
- interactions
- states
- implementation notes
